Welcome to physical-magnitude’s documentation!

The package physical-magnitude allows to perform simple operations with magnitudes including units and uncertainties. It allows to define magnitudes with value, uncertainty and unit. It allows to compute simple operations providing the result not only for the magnitude value, but also its uncertainty and unit. Available operations include summation, subtraction, multiplication and division.

Note

WARNING: This package is under active development. The current version is considered non-stable.
